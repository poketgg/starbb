3. 선택실행구조

- 단순선택구조

if(조건문){ (실행 1) ; } else{ (실행 2); }

- 중첩선택구조

if(조건식 1){ (실행 1) ; } else{ if(조건식 2){ (실행 2-1) ; } else { (실행 2-2) ; } }

4. 반복구조

- for

1) 조건을 아는 경우 :
for(초기화; 조건식; 증감){ (실행할 코드) ; }
2) 횟수를 아는 경우 :
for(실행 1; 실행2; 실행 4){ (실행 3) ; }

- while

while(조건식){ (실행할 코드) ; }

5. 배열

　- 배열

int num[5] = {1, 3, 5, 7, 10} ㅡ> 인덱스는 0부터 시작(따라서, num[1] = 3)

　- 선언

자료형 이름[개수];

　- 초기화

자료형 이름[개수] = {값1, 값2,...};

　- 값 수정(단일로만)

이름[인덱스] = 값;

- 문자열이 들어가는 경우

char name[5] = "Hello" ㅡ> 인덱스 0번(name[0])부터 철자 하나씩 삽입됨.

- 널 문자

값이 없음(존재하지 않음) NULL : stdio.h 라이브러리에 정의되어 있는 명령어.
‘\0’ : 널문자 표현법. 정수 0과 구분해주기 위해 \를 함께 사용함.
0　:　NULL의 아스키 값(정수 0의 아스키 값은 48)
